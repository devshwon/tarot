# P0-02 상태별 QA 체크리스트

각 화면에서 **Loading / Empty / Error / Success** 전환이 의도대로 동작하는지 확인할 때 사용한다.

## 공통

- [ ] 로딩 시 스피너(또는 로딩 UI)가 토큰 기반 크기/색으로 표시되는가?
- [ ] Empty/Error 시 안내 문구가 화면 목적에 맞게 표시되는가?
- [ ] Error 시 "다시 시도" 또는 "돌아가기" 버튼이 동작하는가?

## Home (`/`)

- [ ] **Loading**: 진입 직후 짧게 "오늘의 카드를 불러오는 중…" + 스피너가 보였다가 카드 영역으로 전환되는가?
- [ ] **Success**: 오늘의 카드 뒤집기 → 요약 해석 → 상세 보기/공유 CTA가 정상 노출되는가?
- [ ] (선택) **Error**: 카드 데이터 부재 등 예외 시 안내 + 다시 시도/홈으로 동작하는가?

## Detail (`/detail`)

- [ ] **Loading**: 진입 직후 "상세 해석을 불러오는 중…" 후 잠금 화면 또는 상세 본문으로 전환되는가?
- [ ] **Success (잠금)**: 잠금 해제 CTA → 해제 후 상세 본문이 보이는가?
- [ ] **Success (해제)**: 카드 정보 / 상세 해석 / 조언 / 행운 정보가 모두 표시되는가?
- [ ] **Error**: (예외 시) 다시 시도 / 돌아가기 동작하는가?

## Ask (`/ask`)

- [ ] **Loading**: "카드 뽑기" 클릭 후 "카드를 뽑는 중…"이 잠깐 보였다가 결과로 전환되는가?
- [ ] **Success (폼)**: 질문 입력 → 카드 뽑기 → 결과 카드 + 조언이 표시되는가?
- [ ] **Success (결과)**: "다른 질문하기" 시 폼으로 복귀하는가?

## History (`/history`)

- [ ] **Loading**: 진입 시 "기록을 불러오는 중…" 후 목록 또는 빈 상태로 전환되는가?
- [ ] **Empty**: "아직 기록이 없습니다" + "오늘의 카드 보러가기" 버튼이 동작하는가?
- [ ] **Success**: 최근 14일 기록이 목록으로 표시되는가?
- [ ] **Error**: (저장소 오류 시) 다시 시도 / 홈으로 동작하는가?

## Settings (`/settings`)

- [ ] **Success**: 앱 정보 / 면책 / 데이터 관리가 정상 표시되는가?
- [ ] **Error**: 데이터 초기화 실패 시 "데이터 초기화에 실패했습니다" + 다시 시도 / 설정으로 복귀가 동작하는가?
- [ ] 데이터 초기화 시 `tarot_` prefix 키만 삭제되고, 초기화 후 앱이 재시작되는가?

---

**검증 방법**: 위 체크리스트 순서대로 각 라우트에서 상태 전환을 수동으로 유도한 뒤 동작을 확인한다.
