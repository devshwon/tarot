## 3. 서비스 이용 동작

| # | 항목 | 상태 | 비고 |
|---|------|------|------|
| 1 | 지도 등 꼭 필요한 경우를 제외하고, 제스처 기반 확대·축소 기능은 비활성화돼요. | ✅ | index.html viewport: user-scalable=no, maximum-scale=1, minimum-scale=1 |
| 2 | 미니앱 테마는 라이트 모드로 구현돼 있어요. | ✅ | TDSMobileAITProvider colorPreference 'light', 다크 테마 미사용 |
| 3 | 모든 UI 컴포넌트가 의도한 대로 동작해요. | ✅ | 버튼·탭·카드 뒤집기·공유·광고 플로우 구현 |
| 4 | 스크롤, 터치, 화면 전환 등 인터랙션 반응이 2초 이상 지연되지 않아요. | ⬜ | 실기기/네트워크 환경에서 확인 |
| 5 | 미니앱을 종료했다가 다시 들어와도 필요한 데이터가 유지돼요. | ✅ | localStorage (history, unlocked, gptDetail) 사용 |
| 6 | 서비스 이용에 꼭 필요한 외부 링크가 정상적으로 열려요. | ➖ | 외부 링크 없음(공유 URL은 origin/설정값) |
| 7 | 공유 기능에서 intoss-private:// 사용하지 않고 intoss:// 스킴을 사용해요. | ✅ | share.ts: intoss-private 미사용, getShareUrl()은 origin 또는 VITE_APP_URL |
| 8 | 미니앱 문구에 비속어, 은어, 과도한 유행어가 포함되지 않아요. | ✅ | UI 문구 오락·면책 톤 유지 |
| 9 | 사용자 안내나 확인이 필요한 경우 TDS 모달을 사용해요. | ✅ | Settings: useDialog openConfirm으로 데이터 초기화 확인 |
| 10 | 불법성, 선정성 등 위법한 콘텐츠가 포함돼 있지 않아요. | ✅ | 타로 해석·오락 목적 문구만 사용 |
| 11 | 권한을 요청하기 전에 사용자 동의를 먼저 받아요. | ➖ | 권한 요청 없음 (granite.config permissions: []) |
| 12 | 사용자가 권한을 허용하지 않아도 나머지 기능은 계속 정상적으로 동작해요. | ➖ | 권한 미사용 |
| 13 | 비정상적으로 네트워크/메모리 사용량이 급증하지 않아요. | ⬜ | 실기기·프로파일러로 확인 |
